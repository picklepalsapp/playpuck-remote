PlayPuck BLE Remote

Chrome on Android + HTTPS. Connect, then send score + profiles + Wi-Fi provisioning commands over BLE.

Connection

Not connected Device: — SEQ: 1

Quick Score

Quick Score uses single-letter commands (A / B / U / D / S).

Profile Builder

Sends PROFILE_SELECT,<profileId>,<SEQ> and START_GAME,<SEQ>.

Wi-Fi Setup

Sends WIFI_SET,<ssid>,<pass>,<SEQ> then optionally WIFI_CONNECT,<SEQ>. (Firmware must support these commands.)

Tip: Wi-Fi credentials are sensitive. Don’t use this in public unless you add BLE pairing/encryption.
